# 📖 背单词 - 四级核心词汇

一个双击即用的英语背单词网站，专为大学生备考四级设计，功能类似"不背单词"App。

## ✨ 功能

- **1625 个四级核心词汇** — 完整词库，含中文释义和词性
- **4选1 答题模式** — 随机抽词，答对自动下一题，答错自动收入错题本
- **🔊 发音** — 调用浏览器 Web Speech API，点击喇叭即可朗读
- **📝 错题本** — 自动收集错题，支持复习模式（答对移出）
- **🔍 搜索** — 支持英文和中文搜索单词
- **◀ ▶ 导航** — 答题历史前进后退，随时回顾
- **🎨 毛玻璃 + 视频背景** — 全屏海浪视频 + 毛玻璃卡片效果
- **✨ GSAP 动画** — 悬浮浮起 + 淡入淡出切换

## 🚀 使用方法

1. 双击 `背单词.html`
2. 用 Chrome / Edge / Firefox 打开
3. 开始背单词！

> 视频背景文件较大，未包含在仓库中。如需视频背景，请将 MP4 文件放在同目录下，并在 HTML 中修改 `<source src="...">` 路径。

## 🛠 技术栈

- 纯前端：HTML + CSS + JavaScript（无框架，无 Node.js）
- 动画：GSAP 3.12.5（CDN 加载，断网自动降级）
- 存储：浏览器 localStorage
- 发音：Web Speech API
