# 需求文档 - 背单词网站

## 项目概述
为一个大学生用户开发一个 Windows 上可运行的英语背单词网站，功能类似"不背单词"APP。

## 目标用户
- 大学生（非技术背景）
- 需要背诵四级英语单词

## 功能需求

### FR-01: 单词展示
- 随机从词库中抽取单词
- 以卡片形式展示英文单词
- 卡片风格简洁、直观

### FR-02: 单词发音
- 提供单词读音播放按钮
- 使用浏览器内置语音合成（Web Speech API）
- 点击即播放，无需额外操作

### FR-03: 答题功能
- 提供4个中文翻译选项（1个正确 + 3个干扰）
- 用户选择后判断对错
- 答对：显示绿色反馈，1.5秒后自动跳转下一词
- 答错：显示红色标记正确答案，单词自动加入错题本，用户手动点击继续

### FR-04: 错题本
- 自动收集用户答错的单词
- 提供错题复习页面
- 错题也以选择题形式复习
- 答对的错题可移出错题本
- 数据持久化存储在浏览器中

### FR-05: 单词搜索
- 支持英文单词搜索
- 支持中文翻译搜索
- 实时过滤显示匹配结果

### FR-06: 学习统计
- 显示已背单词数量
- 显示正确率
- 显示错题本中的单词数量

## 非功能需求

### NFR-01: 易用性
- 单个 HTML 文件，双击即可在浏览器打开
- 不需要安装任何软件或依赖
- 界面直观，零学习成本

### NFR-02: 兼容性
- 支持主流浏览器（Chrome、Edge、Firefox）
- Windows 10/11 系统

### NFR-03: 数据安全
- 用户答题数据存储在浏览器本地
- 数据不会丢失（除非清除浏览器缓存）

### NFR-04: 视觉设计
- 淡蓝色背景
- 白色卡片式设计
- 圆角、阴影增加层次感
- 适配不同屏幕尺寸

## 词汇范围
四级核心词汇，约1500词，包含：
- 英文单词
- 音标
- 中文释义
- 词性

## 不包含的功能（当前版本）
- 用户登录/注册
- 云端同步
- 自定义词库导入
- 拼写输入模式（当前仅选择题）
- 间隔重复算法（SRS）
