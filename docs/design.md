# 设计规范 - 背单词网站

## 设计理念
- **简洁**：减少视觉噪音，让用户专注于背单词
- **直观**：无需教程，一眼就知道怎么用
- **友好**：柔和的配色和动画，降低学习焦虑

## 配色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 背景色 | `#e8f4fd` | 淡蓝色主背景 |
| 卡片背景 | `#ffffff` | 纯白卡片 |
| 主文字色 | `#2c3e50` | 深蓝灰 |
| 次要文字 | `#7f8c8d` | 灰色 |
| 正确色 | `#27ae60` | 绿色 |
| 错误色 | `#e74c3c` | 红色 |
| 主按钮色 | `#3498db` | 蓝色 |
| 按钮悬停 | `#2980b9` | 深蓝 |
| 导航激活 | `#1a73e8` | 亮蓝 |
| 阴影色 | `rgba(0,0,0,0.08)` | 轻微阴影 |

## 排版

### 字体
- 英文单词：`'Segoe UI', sans-serif`，加粗
- 中文内容：`'Microsoft YaHei', 'PingFang SC', sans-serif`
- 代码/数据：系统等宽字体

### 字号
| 元素 | 字号 | 字重 |
|------|------|------|
| 英文单词（卡片） | 36px | bold |
| 音标 | 16px | normal |
| 选项文字 | 18px | normal |
| 导航标签 | 16px | 500 |
| 统计数字 | 24px | bold |

## 组件规范

### 导航栏
- 顶部固定，高度 56px
- 白色背景，底部细线分隔
- 三个标签：背单词 | 错题本(N) | 搜索
- 当前页面标签高亮（蓝色下划线）

### 单词卡片
- 白色背景，圆角 16px
- 阴影：`0 2px 12px rgba(0,0,0,0.08)`
- 内边距：40px
- 最大宽度：480px，居中
- 单词居中显示
- 发音按钮在单词右侧（喇叭图标🔊）

### 选项按钮
- 4个选项纵向排列
- 每个选项：圆角 12px，边框 2px solid #e0e0e0
- 悬停：边框变蓝，背景变浅蓝
- 正确态：绿色边框 + 浅绿背景
- 错误态：红色边框 + 浅红背景

### 统计栏
- 卡片底部或页面底部
- 显示：已背 X 词 | 正确率 X% | 错题 X 个
- 小图标 + 数字

### 搜索框
- 顶部搜索输入框，带放大镜图标
- 圆角 24px（胶囊形）
- 输入时实时过滤下方结果列表

## 响应式设计
- 桌面端（>768px）：卡片最大宽度 480px，居中
- 平板端（480-768px）：卡片宽度 90%
- 手机端（<480px）：卡片宽度 95%，字号略小
